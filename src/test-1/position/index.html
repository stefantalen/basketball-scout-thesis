Gebruikerstest ronde 1: Schotpositie

Tijdens deze test worden zes spelsituaties op de laptop getoond. Tussen elke situatie zitten negen seconden pauze.

De bedoeling is dat op het scherm de positie aangegeven wordt vanaf waar de schotpoging(en). Zodra je de positie hebt ingevoerd druk je op de knop "Naar situatie x" waarbij x een getal is.

Voordat je verder gaat informeer bij de instructeur welke variant je moet gebruiken. Zodra je een variant hebt gekozen kun je de video starten.
